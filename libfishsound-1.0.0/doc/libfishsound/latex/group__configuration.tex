\subsection{Platform-\/specific configuration}\label{group__configuration_platforms}
Fish\-Sound can be configured on most platforms using the G\-N\-U autoconf ./configure system described below.

For Win32, see the \doxyref{R\-E\-A\-D\-M\-E.\-win32 }{p.}{group__win32_win32} section. You will need to edit {\ttfamily win32/config.\-h} by hand to achieve the customizations described below.
